# Quilo

Quilo는 학습 자료와 사용자가 직접 만든 데이터를 바탕으로 보고서, 문서, 번역, 분석 작업을 돕는 AI Workspace입니다.

- 서비스: [quilolab.com](https://quilolab.com)
- 운영 상태: 실제 서비스는 별도의 비공개 저장소에서 개발·배포됩니다.

## 이 저장소의 역할

이 저장소는 Quilo의 **공개 제품 개요**입니다. 서비스의 방향, 지원 범위, 고수준 처리 흐름만 공개하며 자체 배포가 가능한 운영 소스 미러가 아닙니다.

공개 범위:

- 제품 소개와 지원 기능
- 민감하지 않은 고수준 아키텍처
- 보안 제보 안내

비공개 범위:

- 서버와 배포 구성
- AI 프롬프트와 보고서 생성 파이프라인
- 데이터베이스 스키마·마이그레이션
- 인증, 결제·크레딧, 관리자 기능
- HWPX/DOCX 렌더링 및 데이터 보정 구현
- 기관별 템플릿과 운영 설정

## 주요 기능

- 화학 사전·결과보고서와 물리 결과보고서 초안 생성
- 사용자 업로드 자료를 우선하는 데이터 기반 작성 흐름
- DOCX/HWPX 문서 출력
- 문서·PDF 번역과 학습용 문서 도구
- 파일, 리서치, 데이터 분석을 연결하는 작업 공간

구현 세부사항은 의도적으로 생략했습니다. 제품 수준의 흐름은 [아키텍처 개요](docs/architecture.md)에서 확인할 수 있습니다.

## 문의와 보안

일반 문의는 [Quilo 서비스](https://quilolab.com)의 공식 문의 채널을 이용해 주세요. 보안 문제는 공개 이슈에 개인정보나 재현용 사용자 데이터를 올리지 말고 [보안 정책](SECURITY.md)을 따라 제보해 주세요.

## 라이선스

이 저장소에 실제로 포함된 공개 문서와 자료에만 루트의 `LICENSE`가 적용됩니다. 비공개 운영 구현, 브랜드 자산, 사용자 데이터 및 기관별 양식은 이 저장소의 공개 범위에 포함되지 않습니다.
