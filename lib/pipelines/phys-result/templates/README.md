# Physics Result HWPX Template

The public repository does not include institution-specific `.hwpx` report
templates.

To enable template-based physics HWPX output, place a template you have
permission to use at:

```text
lib/pipelines/phys-result/templates/result-report-template.hwpx
```

If the file is missing, the generator falls back to creating a generic HWPX
document from scratch.
