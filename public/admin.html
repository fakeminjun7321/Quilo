사용자 계정 상태 크레딧 이번 시간 사용량 가입일 조작

API 키 위임

지정한 사용자에게 일정 기간 동안 관리자 키로 무료 사용할 권한을 줍니다. 위임 기간에는 그 사용자의 보고서 생성과 파일 챗봇이 크레딧 차감 없이 서버 키로 실행됩니다. 개인 API 키를 저장하지는 않습니다. (DB: api_key_grants — 마이그레이션 필요)

Pro 회원

Pro 회원은 모든 Pro 기능(무료 보고서 6종 · 파일 챗봇 · 창작/바이브/물리 스튜디오 · Quilo Code · 상대론 공부)을 기능별 일일 한도 안에서 무료로 사용합니다. 내부적으로 pro 우산 키 하나로 관리합니다 — 기능별 개별 테스터 지정은 더 이상 쓰지 않습니다.

임시 공개

선택한 기능을 정해진 시간 동안 선택한 대상 등급(모두/Pro/Max)에게 개방합니다(체험 이벤트용). 일일 한도는 그대로 적용되고, 시간이 지나면 자동으로 닫힙니다. 0시간 = 즉시 해제.

Max 구독 — 백그라운드·PDF 통번역

지정한 사용자에게 일정 기간 Max 권한(백그라운드 실행 + PDF 통번역 월 300쪽)을 줍니다. 켜진 사용자는 보고서를 제출하고 탭/창을 닫아도 서버가 끝까지 생성하며, '내 파일'과 완료 이메일로 받을 수 있습니다. (30일 5,000원 · 입금 승인 시 10크레딧 자동 지급 · DB: background_subscriptions)

입금 신청 대기

사용자가 입금 후 'Max 신청'을 하면 여기에 뜹니다. 입금 확인 후 승인하면 즉시 Max(백그라운드 실행)이 활성화됩니다. (기본 30일 · 활성이면 만료일에서 연장)

학교 도입 신청

외부 학교가 /school-apply.html 에서 도입을 신청하면 여기에 뜹니다. 학생 이메일 도메인·학번 체계·업로드한 학교 양식으로 그 학교 전용 설정을 준비하세요. 상태를 바꿔 진행 상황을 관리할 수 있습니다.

Pro 기능

Pro 기능을 켜고/끄고, 지정한 Pro 회원 계정에게만 노출합니다. 관리자는 항상 사용 가능. 여러 Pro 기능을 동시에 운영할 수 있습니다.

문제집 메이커 한도

문제집 메이커가 한 번에 만들 수 있는 최대 문제 수입니다. 사용자가 올린 파일에서 이보다 많은 문제가 추출되면 생성이 거부됩니다. 관리자는 이 한도를 받지 않습니다. 0 = 무제한.

보고서 생성 로그

최근 30일 서비스 품질 집계와 비용 로그입니다. 분석 화면에는 보고서 제목·본문·업로드 파일명이 포함되지 않습니다.

소명·제재
