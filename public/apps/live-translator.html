음성 인식·번역 로컬 처리

듣고, 인식하고, 자막으로

마이크 입력과 시스템 오디오를 받아 발화 단위로 음성을 인식하고, 로컬 번역 모델을 거쳐 항상 최상단 자막 오버레이에 표시하도록 설계돼 있습니다.

필요한 모델을 처음 내려받은 뒤 음성 인식과 번역은 컴퓨터 안에서 처리됩니다. 별도의 API 키를 요구하지 않는 구성을 기준으로 합니다.

  1. 01

    오디오 입력

    마이크 또는 시스템에서 재생되는 소리를 입력으로 받습니다.

  2. 02

    로컬 처리

    Whisper 기반 음성 인식과 로컬 언어 모델 번역을 순서대로 실행합니다.

  3. 03

    자막 표시

    번역한 문장을 다른 앱 위에 놓이는 자막 오버레이로 보여줍니다.

확인된 플랫폼 구성

앱 패키지 설정에서는 macOS Apple Silicon용 DMG와 Windows x64용 설치 프로그램을 대상으로 합니다. 첫 실행 시 필요한 음성·번역 모델을 내려받는 흐름이 포함돼 있습니다.
