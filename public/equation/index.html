한글 수식 변환기

이 컴포넌트의 구현은 현재 Quilo 오픈소스 배포본에 포함되지 않습니다.

왜 제외되었나요?

기존 브라우저 변환기는 비영리 사용 조건이 붙은 제3자 코드를 포함합니다. 이 조건은 OSI 오픈소스 정의와 맞지 않아, 재라이선스 허락을 받거나 독립 구현으로 교체하기 전까지 공개 소스에서 분리했습니다.

서버의 HWPX 생성 경로는 공개된 자체 변환기로 폴백할 수 있습니다. 공개본에서 누락된 권리 제한 파일을 복원하지 말아 주세요.

독립 구현, 테스트 코퍼스 설계 또는 라이선스 검토를 도와주실 분은 GitHub Issues에서 의견을 남겨 주세요.
