이용약관

시행일: 2026년 7월 15일 · 최종 변경: 2026년 7월 15일

1. 서비스의 성격

Quilo는 보고서 초안 생성, 학습·연구 보조, 문서 변환, 창작·코딩, 파일·외부 서비스 연동을 제공하는 AI Workspace입니다. 과제 대행, 제출용 완성본 판매, 학교 평가 우회 서비스가 아니며, 별도 표시가 없는 한 학교·교육청·연구기관·출판사와 공식 제휴된 서비스가 아닙니다.

서비스가 생성한 결과물은 제출용 완성본이 아니며, 사용자는 내용을 직접 검토·수정하고 사실관계, 계산, 출처, 학교 규정을 확인해야 합니다.

다만 생성 결과에 내용상 오류가 확인되는 경우 환불정책에 따라 환불 또는 크레딧 복구를 요청할 수 있습니다.

2. 이용 조건

  • 만 14세 미만 사용자는 서비스를 이용할 수 없습니다.
  • 사용자는 본인이 이용 권한을 가진 실험 매뉴얼, 데이터, 사진, 파일만 업로드해야 합니다.
  • 타인의 개인정보, 민감정보, 비공개 자료, 저작권 침해 우려가 있는 자료를 무단 업로드하면 안 됩니다.
  • 생성 결과물을 그대로 제출하거나, 학칙·평가기준·저작권을 우회하기 위한 목적으로 사용하면 안 됩니다.

3. 학칙 및 학업윤리

  • 사용자는 생성 전 학교, 교사, 수업, 대회, 평가기준의 AI 사용 허용 범위를 직접 확인해야 합니다.
  • 학교나 교사가 AI 사용 사실 표시, 출처 표시, 사용 범위 제한을 요구하는 경우 사용자는 그 기준을 따라야 합니다.
  • 생성 결과물을 그대로 제출하거나, 본인이 작성한 것처럼 속이거나, 표절·부정행위·평가기준 회피 목적으로 사용하면 안 됩니다.
  • 운영자는 학칙 위반, 저작권 침해, 과제 대행성 사용이 의심되는 경우 생성 제한, 계정 제한, 파일 삭제, 필요한 최소 기록 보관을 할 수 있습니다.

4. AI 생성물 고지

  • AI가 만든 내용에는 사실 오류, 계산 오류, 출처 누락, 부정확한 해석이 포함될 수 있습니다.
  • 보고서의 최종 사용, 제출, 인용, 수정 여부에 대한 책임은 사용자에게 있습니다.
  • 서비스는 점수, 평가 결과, 학교 제출 가능성, 표절 판정 회피를 보장하지 않습니다.
  • 내용상 오류가 있는 경우 사용자는 오류 위치와 내용을 제시해 환불 또는 크레딧 복구를 요청할 수 있습니다.

5. 업로드 파일과 저작권

  • 사용자는 업로드 자료를 서비스에 입력하고 보고서 초안 생성에 사용할 권한이 있음을 확인합니다.
  • 서비스는 학교 매뉴얼이나 사용자 업로드 파일을 공개 배포하지 않습니다.
  • 업로드 파일은 보고서 생성 목적으로만 사용하며, 생성 파일은 사용자별 최대 3개, 최대 24시간 임시 보관합니다.
  • Quilo는 업로드 자료와 생성 결과를 자체 AI 모델 학습 데이터셋이나 별도 공유 DB로 재사용·판매하지 않습니다.
  • 선택한 AI 기능을 수행하기 위해 필요한 범위에서 입력이 Anthropic, OpenAI, Google, Groq, Mistral AI 등 해당 제공자의 API로 전송될 수 있으며, 제공자 API 약관과 정책이 함께 적용됩니다.
  • 권리자 또는 학교의 문제 제기가 있는 경우 운영자는 관련 파일을 삭제하고 계정을 제한할 수 있으며, 분쟁 대응에 필요한 최소 로그만 보관할 수 있습니다.
  • 사용자가 권한 없는 자료를 업로드해 발생하는 저작권·계약·학칙상 문제는 사용자 책임입니다.

6. 서비스 운영 기록과 선택형 이용 분석

  • Quilo는 장애 대응, 보안, 부정 사용 방지, 정산과 품질 검증을 위해 작업 상태·처리 시간·오류 코드·보고서 종류·모델·형식·업로드 개수와 확장자·용량 구간·다운로드 여부 등 최소 운영 기록을 처리합니다.
  • 운영 분석 기록에는 업로드 파일명·내용, 사용자 메모, 보고서 제목·본문, 인증정보를 넣지 않습니다.
  • 기능 선택부터 생성·미리보기·다운로드까지의 이용 흐름 분석은 계정 센터에서 별도로 동의한 사용자에게만 적용됩니다.
  • 선택형 분석 동의 여부는 핵심 기능·크레딧·서비스 품질에 영향을 주지 않으며 언제든 철회할 수 있습니다.
  • 사용자가 결과 평점과 문제 태그를 제출하면 해당 코드형 평가를 제품 개선에 이용할 수 있습니다. 자유서술 보고서 내용은 품질평가 항목에 포함하지 않습니다.

7. 계정, API 키와 외부 연동

  • 사용자는 계정과 비밀번호, 발급받은 API 토큰, BYOK 키를 안전하게 관리해야 하며 타인과 공유하면 안 됩니다.
  • BYOK 키와 OAuth 토큰은 사용자가 요청한 제공자 호출과 연동에만 사용합니다. 사용자는 언제든 계정 센터 또는 해당 외부 서비스에서 연결을 해제할 수 있습니다.
  • Dropbox·Google Drive/Docs·Notion 등에 저장한 파일의 보관·삭제·공유 권한에는 해당 외부 서비스의 약관과 설정이 적용됩니다.
  • 외부 API 토큰의 권한 범위, 요청 제한, 비용과 결과물 책임은 개발자 문서와 본 약관을 함께 따릅니다.

8. 크레딧과 유료 이용

  • 보고서 생성에는 종류별 크레딧이 차감될 수 있습니다.
  • 화면에 표시되는 예상 비용은 추정치이며, 실제 차감 기준은 서비스에 고지된 크레딧 단가를 따릅니다.
  • 현재 크레딧은 제한된 사용자에게 관리자 수동 발급 방식으로 운영되며, 공개 자동 결제나 불특정 다수 대상 판매 기능을 제공하지 않습니다.
  • 공개 유료 판매를 시작하는 경우 판매자 정보, 연락처, 가격, 결제 방법, 청약철회·환불 기준, 소비자 피해 처리 방법을 사전에 고지하고, 필요한 경우 사업자등록, 통신판매업 신고, 세금 처리를 진행합니다.

9. 금지 행위

  • 타인 계정 사용, 계정 공유, 비밀번호 유출
  • 과도한 자동 요청, 보안 우회, 파일 제한 우회
  • 타인의 API 키·OAuth 연결 사용, 토큰 권한 우회, 운영·분석 시스템에 허위 이벤트를 반복 전송하는 행위
  • 악성 파일, 압축 폭탄, 저작권 침해 파일 업로드
  • 서비스 결과를 학교 규정 위반, 표절, 부정행위, 과제 대행, AI 사용 사실 은폐에 사용하는 행위

10. 책임 제한

서비스는 학습 보조 도구로 제공되며, 점수, 평가 결과, 학교 제출 가능성, 표절 판정 회피, 계속적 이용 가능성을 보장하지 않습니다. 사용자의 제출, 평가, 징계, 저작권 분쟁, 자료 오남용으로 인한 책임은 사용자에게 있습니다.

단, 서비스 운영자의 고의 또는 중대한 과실로 인한 법률상 책임, 법령상 제한할 수 없는 소비자 권리, 환불정책에 따른 환불 또는 크레딧 복구 권리는 배제하지 않습니다.

11. 이용 중단과 자료 삭제

운영자는 보안 사고, 법령 위반, 저작권 침해, 권한 없는 자료 업로드, 계정 공유, 과도한 자동 요청이 확인되면 해당 계정의 이용을 제한하거나 관련 파일을 삭제할 수 있습니다. 필요한 경우 위반 사실 확인을 위한 최소한의 기록을 보관할 수 있습니다.
